# Requirements

This setup depends on the following tools. Install each from its official source.

## Core Tools

- Kitty (terminal emulator)  
  https://sw.kovidgoyal.net/kitty/

- Neovim  
  https://neovim.io/

- Zsh  
  https://www.zsh.org/

- Zinit (Zsh plugin manager)  
  https://github.com/zdharma-continuum/zinit

- starship (prompt)  
  https://starship.rs/

- Yazi (file manager)  
  https://yazi-rs.github.io/

- fzf (fuzzy finder)  
  https://github.com/junegunn/fzf

- bat / batcat (cat replacement with syntax highlighting)  
  https://github.com/sharkdp/bat

- zoxide (smarter cd)  
  https://github.com/ajeetdsouza/zoxide

- tmux (terminal multiplexer)  
  https://github.com/tmux/tmux/wiki

- TPM (Tmux Plugin Manager)  
  https://github.com/tmux-plugins/tpm

---

## Notes

- On Debian/Ubuntu, `bat` may be installed as `batcat`.
- Some tools may also be available via your system package manager (`apt`, `dnf`, `pacman`, etc.).
- It is recommended to install the latest stable versions where possible.
